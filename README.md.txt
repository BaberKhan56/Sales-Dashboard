# Power BI Sales Dashboard

This project demonstrates basic data analysis and visualization using Power BI.

## Project Features
- Connecting Excel data to Power BI
- Data preparation and cleaning
- Data aggregation using statistical functions (SUM, AVERAGE, COUNT)
- Data visualization using:
  - Bar Chart
  - Pie Chart
  - Line Chart
- Creating a basic interactive dashboard

## Tools Used
- Power BI Desktop
- Microsoft Excel

## Dashboard Insights
The dashboard helps analyze sales performance by region, category, and time using different visualizations.

## Files Included
- Sales_Dashboard.pbix – Power BI dashboard file
- sales_data.xlsx – Dataset used for analysis
