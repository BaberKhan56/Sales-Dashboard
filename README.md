# Power BI Sales Dashboard

This project demonstrates how to connect data, prepare it for analysis, apply aggregation functions, and create visualizations using Power BI.

## 📊 Project Overview

The goal of this project is to analyze sales data and present insights using interactive visualizations and a basic dashboard in Power BI.

## ⚙️ Tools Used

- Power BI Desktop
- Microsoft Excel
- GitHub
- VS Code

## 📂 Dataset

The dataset used in this project contains the following fields:

- OrderID
- Date
- Region
- Product
- Category
- Sales
- Quantity
- Profit

The dataset was imported from an Excel file and prepared for visualization in Power BI.

## 🔄 Data Preparation

The following steps were performed:

- Connected Excel data to Power BI
- Checked and corrected data types
- Cleaned and prepared data for analysis
- Removed unnecessary data and ensured consistency

## 📈 Data Aggregation & Statistical Functions

Power BI aggregation functions were used to summarize the dataset:

- **SUM** – Total Sales and Quantity
- **AVERAGE** – Average Profit
- **COUNT** – Number of Orders
- **MAX / MIN** – Highest and Lowest values

These aggregations help in understanding overall sales performance.

## 📊 Data Visualizations

The following visualizations were created:

- **Bar Chart** – Sales by Region
- **Pie Chart** – Sales by Category
- **Line Chart** – Sales Trend Over Time
- **Card Visual** – Total Sales and key metrics

These visualizations help to analyze patterns and compare sales performance.

## 📊 Dashboard

A basic Power BI dashboard was created to display key insights including:

- Total Sales
- Average Profit
- Sales distribution by region
- Category-wise sales
- Sales trends over time

The dashboard provides a clear and interactive view of the sales data.
